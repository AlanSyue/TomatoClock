## TomatoClock (蕃茄鐘)

## Setup

This project uses the following tools. Go check them out if you don't have them locally installed.

- [node、npm](https://nodejs.org/en/)

```
git clone https://github.com/Nick0603/TomatoClock.git
cd TomatoClock
npm install
npm run start
```
